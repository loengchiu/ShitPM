# Artifact Schema

## Project Brief

核心字段：
- 项目背景
- 范围边界
- 用户与角色
- 关键流程
- 关键规则
- 需求变更记录
- 风险与待确认项

## Feature List

核心字段：
- 模块
- 涉及角色
- 功能点编号
- 功能点名称
- 说明
- 优先级
- 版本
- PRD章节
- 版本边界（可附加）
- 关键需求摘要（可附加）

## Page Structure

核心字段：
- 页面编号
- 页面名称
- 页面目标
- 区域名称
- 对应功能点
- 作用
- 关键操作
- 关键状态

## PRD

核心字段：
- 上游锚点
- 项目背景
- 范围边界
- 用户与角色
- 关键流程
- 关键规则
- 详细需求说明
- 风险与待确认项
- 模块涉及角色落位
- 版本边界落位

## Prototype

核心字段：
- 页面名称
- 页面目标
- 关键区域
- 关键状态
- 关键流程
- 视觉基线
- 页面与区域回溯关系

## Prototype Annotation

核心字段：
- 需求编号
- 模块名称
- 页面定位
- 角标挂载点
- 浮窗标题
- 浮窗正文
- 编号回写位置（可选）
